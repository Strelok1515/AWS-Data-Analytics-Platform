# AWS Data Analytics Platform

# 

## 1.Overview

# 

# Designed and implemented a cloud-based data analytics platform using Amazon Web Services (AWS), Amazon Redshift, SQL, and Tableau.

# 

# The project demonstrates an end-to-end analytics workflow including cloud storage, data warehousing, SQL-based analysis, and business intelligence reporting.

# 

## 2.Technologies

# 

# \* Amazon S3

# \* Amazon Redshift

# \* AWS IAM

# \* SQL

# \* Tableau

# \* Data Warehousing

# \* ETL Workflows

# 

## 3.Architecture

# 

# Data is uploaded to Amazon S3 and loaded into Amazon Redshift using SQL COPY operations. The warehouse is then queried using SQL and connected to Tableau through a live data connection for reporting and dashboard visualisation.

# 

## 4.Key Features

# 

# \* Cloud-based data storage using Amazon S3

# \* Amazon Redshift data warehouse

# \* SQL data loading and querying

# \* Tableau dashboard visualisation

# \* IAM role configuration

# \* Security group configuration

# \* Data analytics and reporting

# 

## 5.Project Outcomes

# 

# \* Processed and analysed over 8,000 weather sensor records

# \* Built an end-to-end cloud analytics workflow

# \* Generated visual insights using Tableau dashboards

# \* Applied cloud architecture and data warehousing concepts

#
